# はじめての使い方ガイド 🌱

「つづく」を動かすための、プログラミングが初めての方向けの手順書です。
専門用語は最小限にして、順番どおりにやれば必ず動くように書いています。
（お使いのパソコンは Windows を想定していますが、Mac の場合の補足も入れています）

---

## この作業でやることの全体像

料理でたとえると、こんな流れです。

1. **調理器具をそろえる** … Node.js（ノードジェイエス）というソフトを入れる ← 最初の1回だけ
2. **材料を用意する** … `npm install` で必要な部品をダウンロード ← 最初の1回だけ
3. **調理する** … `npm run dev` でアプリを起動する ← 毎回これ
4. **食べる** … ブラウザで開いて使う

最初の準備（1と2）が終われば、次からは 3 と 4 だけで動きます。

---

## ステップ1：Node.js を入れる（最初の1回だけ）

「つづく」は Node.js というソフトの上で動きます。まずこれをインストールします。

1. ブラウザで **https://nodejs.org/ja** を開く
2. 「**LTS**」と書かれた方の緑色のボタンをクリック（LTS＝安定版という意味。こちらを選べば安心です）
3. ダウンロードされたファイルをダブルクリック
4. インストール画面が出たら、基本的に「**次へ（Next）**」を押していけばOK。途中の設定は変えなくて大丈夫です
5. 最後に「**Finish（完了）**」を押す

これで調理器具がそろいました。

> 💡 Mac の方も同じく https://nodejs.org/ja から LTS 版を入れてください。

---

## ステップ2：「コマンドを打つ画面」を開く

聞き慣れないかもしれませんが、パソコンに文字で命令を出すための画面を使います。怖くありません。

### Windows の場合
1. キーボードの **Windowsキー** を押す
2. `powershell` と入力する
3. 出てきた「**Windows PowerShell**」をクリックして開く

### Mac の場合
1. **command + スペース** を押して検索を開く
2. `terminal` と入力する
3. 「**ターミナル**」をクリックして開く

黒い（または青い）文字だけの画面が出れば成功です。この画面を「ターミナル」と呼びます。

---

## ステップ3：アプリのフォルダに移動する

ターミナルに、これから使う「つづく」のフォルダの場所を教えます。

一番かんたんな方法：

1. ターミナルに `cd ` と打つ（**cd のあとに半角スペースを1つ**入れます。まだ Enter は押さない）
2. `tsuzuku` フォルダを、ターミナルの画面の上に**マウスでドラッグ＆ドロップ**する
   → フォルダの場所（住所のような文字列）が自動で入力されます
3. **Enter キー**を押す

うまくいくと、行の先頭の表示が `tsuzuku` を含むものに変わります。これで「今このフォルダにいます」という状態になりました。

> 💡 ドラッグがうまくいかない場合は、`cd ` のあとにフォルダの場所を直接キーボードで入力してもOKです。

---

## ステップ4：必要な部品を入れる（最初の1回だけ）

ターミナルに次のとおり打って、**Enter** を押します。

```
npm install
```

たくさんの文字が流れて、少し時間がかかります（数十秒〜数分）。
文字が止まって、また入力できる状態に戻れば完了です。
（黄色い警告メッセージが出ることがありますが、エラーでなければ気にしなくて大丈夫です）

これは最初の1回だけでOK。次回以降は飛ばせます。

---

## ステップ5：アプリを起動する

ターミナルに次を打って、**Enter**。

```
npm run dev
```

すると、こんな表示が出ます（数字は違うことがあります）：

```
  VITE v5.4.21  ready in 300 ms

  ➜  Local:   http://localhost:5173/
```

この `http://localhost:5173/` が、アプリのある場所です。

---

## ステップ6：ブラウザで開いて使う

1. Chrome や Edge などのブラウザを開く
2. アドレス欄に `http://localhost:5173/` と入力して Enter
   （またはターミナル上のリンクを **Ctrl を押しながらクリック**）

「つづく」の画面が出れば成功です！🎉

### 基本の使い方
- 習慣の**大きな四角ボタンを押す**と、今日の達成として記録されます（もう一度押すと取り消し）
- 下の**曜日ボタン**で、今週の他の日もチェックできます
- 「**年間の記録を見る ▼**」を押すと、1年ぶんの達成マップが開きます
- 右上の「**＋ 習慣を追加**」で、自分の習慣を作れます（名前・絵文字・色を選べます）
- 右上の**太陽/月アイコン**で、明るい画面と暗い画面を切り替えられます
- 記録は自動でパソコンに保存されるので、閉じても消えません

---

## 終わり方・次回の起動

### 終わるとき
ターミナルの画面で **Ctrl + C** を押すと、アプリが止まります。
（ブラウザのタブは普通に閉じてOK）

### 次回また使うとき
準備（ステップ1・2・4）はもう不要です。次の3つだけ：

1. ターミナルを開く（ステップ2）
2. フォルダに移動する（ステップ3）
3. `npm run dev` を打つ（ステップ5）→ ブラウザで開く

---

## 困ったときは（トラブル対処）

**`npm` は認識されていません、と出る**
→ Node.js が正しく入っていないか、ターミナルの再起動が必要です。ターミナルを一度閉じてから開き直してください。それでもダメならステップ1をやり直します。

**`http://localhost:5173/` を開いても真っ白**
→ ターミナルで `npm run dev` が動いたままになっているか確認してください。止まっている場合はもう一度 `npm run dev` を実行します。

**ポート 5173 が使われています、と出る**
→ 別の番号（例：5174）で起動します。ターミナルに表示された新しいアドレスを開いてください。

**`npm install` でエラーが出る**
→ インターネットに接続されているか確認し、もう一度 `npm install` を試してください。

---

困ったら、エラー画面の文字をそのまま伝えてもらえれば、一緒に解決します。
